About

This website provides a way for students with four functionalities to find the best unit in UWA.

1. Registration: You will need an account to vote. Please use several seconds to register one! Thanks!

2. Log in: You can log in to the website with your credential. You can reset your password in the log in page if you forgot yours. You can also change your username and email in MyAccount.

3. Vote: This section requires users being logged in. You can simply input the unit code in UWA and their preference for the unit.

4. Result: The result is according to the preferences contributed by all participants. It simply get the average of each unit and rank them in descending order. The results will only display for those units with more than three voters.
