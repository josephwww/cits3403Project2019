About

This website provides a way for students to find the best unit in UWA.

1. Registration: Generally, in order to ensure the reliability of the data, every participant need to register their own account. Once registered, user can change their account details including email and username in the account page.

2. Log in: User logs in to the website with their own credential, otherwise they will be introduced to registration page.

3. Vote: This section requires users being logged in. Users simply input the unit code in UWA and their preference for the unit. The unit code occupies eight characters and the website would check the validity of the unit code. You can only vote for the same unit once or change your preference for the unit by inputting the same unit code and new preference.

4. Result: The result is according to the preferences contributed by the participants. It simply get the average of each unit and rank it in descending order. Everyone has access to the result page. This page provides the view of all units have been voted by preference in descending order with how many participants. Additionally, The bar graph under each unit shows the data in an easy way.
